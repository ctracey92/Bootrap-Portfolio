Train Scheduler

A train schedulng app. The user inputs the train name, destination, first arrival time, and how often it runs. Based on the user input the application will calculate the number of minutes until the next train arrival, and what time it will arrive at. The input, and calculated variables will be stored in firebase so that it will be loaded on every use of the page.
